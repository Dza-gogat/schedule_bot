# Telegram-бот

Простой бот с командами и эхо-ответом.

## Как запустить

1. **Получить токен**
   - Открой Telegram, найди [@BotFather](https://t.me/BotFather)
   - Отправь `/newbot`, придумай имя и username бота
   - Скопируй выданный токен

2. **Установить зависимости**
   ```bash
   pip install -r requirements.txt
   ```

3. **Задать токен** (один из способов):
   - **Вариант А.** Создай файл `.env` по образцу `.env.example` и впиши туда токен  
     (если используешь библиотеку `python-dotenv`, добавь её в `requirements.txt` и в начале `bot.py`: `from dotenv import load_dotenv; load_dotenv()`).
   - **Вариант Б.** В PowerShell перед запуском:
     ```powershell
     $env:TELEGRAM_BOT_TOKEN = "твой_токен_сюда"
     python bot.py
     ```

4. **Запуск**
   ```bash
   python bot.py
   ```

## Команды бота

- `/start` — приветствие
- `/rasp` — расписание на **сегодня** (день берётся по реальному времени)
- `/rasp пн`, `/rasp вт`, `/rasp ср`, `/rasp чт`, `/rasp пт`, `/rasp сб`, `/rasp вс` — расписание на выбранный день
- `/week` — выбрать неделю (I или II); выбор сохраняется до следующего изменения
- `/help` — справка

Токен никому не показывай и не загружай в репозиторий.
